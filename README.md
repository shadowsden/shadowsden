# Shadow's Den 🌙

My personal website — built with plain HTML, CSS, and JavaScript. Hosted on GitHub Pages.

## Setup

1. Create a GitHub account at github.com
2. Create a new repository called `shadowsden`
3. Upload all these files to the repository
4. Go to **Settings → Pages** and set Source to **GitHub Actions**
5. Your site will be live at `https://yourusername.github.io/shadowsden`

## Adding Photos to the Gallery

1. Add your photos to the `/images` folder in the repository
2. Push to GitHub
3. The GitHub Action automatically updates the gallery order (newest first)

## Editing the About Me Page

Open `about.html` and look for the `✏️ Edit this:` comments — they mark exactly where to change your text.

## Adding a Book

Open `books.html` and follow the instructions in the `BOOK TEMPLATE` comment section.

## File Structure

```
shadowsden/
├── index.html        ← Home page
├── books.html        ← Books page
├── gallery.html      ← Gallery page
├── about.html        ← About me page
├── contact.html      ← Contact page
├── style.css         ← All styles
├── components.js     ← Shared nav + footer
├── images/           ← Add your photos here
│   └── index.json    ← Auto-generated by GitHub Action
└── .github/
    └── workflows/
        ├── deploy.yml          ← Auto-deploys site to GitHub Pages
        └── generate-index.yml  ← Auto-generates gallery order
```
